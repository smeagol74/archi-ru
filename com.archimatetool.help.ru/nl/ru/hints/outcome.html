Концепт Выходной результат представляет конечный результат, который получен каким-либо образом. Этот концепт позволяет обзорно моделировать производственные результаты применения способностей предприятия. Выходной результат тесно связан с целями, требованиями и другими намерениями. Определение требований, целей, проектирование способностей предприятия происходит в терминах Выходных результатов.

Именование Выходных результатов должно определять конечный результат, к которому требуется придти для того, чтобы не путать их с целями, для этого при именовании концепта должен использоваться глагол совершённой формы и существительное.
