Элемент типа Инструмент используется для моделирования машин, вспомогательных средств и другого производственного обеспечения, которое можно использовать при обработке материалов. Инструмент всегда обладает физическим воплощением.

Понятие Инструмент охватывает все элементы активной структуры, которые выполняют производственные процессы обработки или использования материалов. Инструмент является частным случаем узла (связан с узлом отношением специализации). Таким образом становится возможным моделирование узлов, которые образованы совокупностью элементов ИТ-инфраструктуры (устройства, системное программное обеспечение) и других типов инфраструктурных элементов (например аппарат МРТ в лечебном учреждении).

Инструмент может обладать Доступом к Материалам (например: создавать, использовать, хранить, перемещать, преобразовывать). Инструмент может обслуживать другой Инструмент, Производственную роль, Производственного деятеля. Фабрика может быть назначена, как использующая Инструмент. Инструмент может состоять из других инструментов. Инструмент может быть назначен на фабрику, (например как используемый там), и может быть аггрегирован в каком-то размещении.

Для именования элементов типа Инструмент следует использовать имена существительные в именительном падеже.

Категория элемента языка:

Физический. Активная структура.

Примеры:

Томограф, Сборочная линия, 3D-принтер.
