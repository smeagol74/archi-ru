Использование Archi

При работе с данным приложением, вам может понадобиться общее представление о том, как тут всё устроено.

Окна и вкладки

Основное поле редактирования описаний находится в части окна приложения посередине справа. Окна и вкладки можно перетаскивать и перетасовывать в соответствии с вашими потребностями, в том числе выносить их в отдельные "плавающие окна".

Размещение окон можно вернуть к исходному, выбрав "Сбросить расположение окон" в меню "Окно". Сброс всех настроек на исходные выполняется командой "Сбросить к значениям по-умолчанию" из меню "Помощь". Пользовательские настройки при этом сохраняются в обоих случаях.


Undo/Redo

Команды Undo и Redo доступны для всех действий в Archi выполняемых пользователем.

Стэк команд закреплён за каждой моделью отдельно и команда вызывается для модели выбранной в дереве моделей. Щелчок мышкой на объекте типа "Описание" или "Модель позволит использовать данные команды, если для модели выполнялись какие-либо действия.


"Шпаргалки"

Шпаргалки содержат готовые пошаговые рецепты, для достижения некоторых целей. Некоторые шаги могут исполняться непосредственно "шпаргалкой", для других есть подробное описание действий пользователя. Шпаргалки доступны через основное меню "Помощь". На данный момент в поставку Archi входят 2 шпаргалки: "Create a Map View" и "Create a New Model".


Getting Help

Contextual and full help is available from the main "Help" menu. On Windows pressing the "F1" key will invoke contextual help.
